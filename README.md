# Customer Behaviour Analysis

## Overview
This project focuses on analyzing customer purchasing behaviour using Python, SQL, PostgreSQL, and Power BI. The workflow includes data loading, cleaning, exploratory data analysis (EDA), SQL-based business analysis, dashboard creation, and report generation.

The main goal of this project is to identify customer trends, purchasing patterns, revenue insights, and subscription behaviour to support business decision-making.

---

## Dataset
The dataset contains customer shopping and purchasing information, including:

- Customer ID
- Age
- Gender
- Category
- Item Purchased
- Purchase Amount
- Review Rating
- Shipping Type
- Subscription Status
- Discount Applied
- Previous Purchases
- Season
- Location

Total Records: **3900+**

## Project Workflow

### 1. Data Loading
- Loaded CSV dataset into Python using Pandas.

### 2. Data Cleaning
- Checked missing values
- Removed duplicates
- Fixed column formats
- Created new derived columns

### 3. Exploratory Data Analysis (EDA)
Performed:
- Statistical analysis
- Revenue analysis
- Customer segmentation
- Age group analysis
- Subscription analysis
- Shipping analysis

### 4. PostgreSQL Database Integration
- Connected Python to PostgreSQL using SQLAlchemy

Executed multiple SQL queries to answer business questions such as:
- Revenue by gender

### 6. Power BI Dashboard

## Key Insights

- Clothing category generated the highest revenue.
- Young adults contributed the highest sales.
- Subscribed customers showed strong purchasing behaviour.
- Express shipping users spent more on average.
- Accessories and clothing were the top-performing categories.

## How to Run the Project

### 1. Clone Repository

```bash
git clone https://github.com/your-username/customer-behaviour-analysis.git
```

### 2. Install Required Packages

```bash
pip install -r requirements.txt
```

### 3. Run Jupyter Notebook

```bash
jupyter notebook
```

### 4. Connect PostgreSQL
Update database credentials in Python script:

```python
username = "postgres"
password = "your_password"
host = "localhost"
port = "5432"
database = "customer_behaviour"
```

### 5. Open Power BI Dashboard
Open:

```text
customer_behaviour.pbix
```
